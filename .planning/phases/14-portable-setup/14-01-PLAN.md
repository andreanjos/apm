       **`encode(setup: &PortableSetup) -> anyhow::Result<String>`:**
       - Step 1: `serde_json::to_vec(setup)?` (compact JSON, no pretty-print)
       - Step 2: `DeflateEncoder::new(Vec::new(), Compression::best())` — write all, finish
       - Step 3: `URL_SAFE_NO_PAD.encode(&compressed)` from `base64::engine::general_purpose`
       - Return `format!("{SCHEME_PREFIX}{b64}")`
       - Use: `use base64::engine::{general_purpose::URL_SAFE_NO_PAD, Engine};`
       - Use: `use flate2::write::DeflateEncoder; use flate2::Compression;`

       **`decode(input: &str) -> anyhow::Result<PortableSetup>`:**
       - Strip `SCHEME_PREFIX` or bail with "missing 'apm1://' prefix"
       - Step 1: `URL_SAFE_NO_PAD.decode(payload)?`
       - Step 2: `DeflateDecoder::new(&compressed[..])` — read_to_end
       - Step 3: `serde_json::from_slice(&json)?`
       - Validate `setup.v == 1` or bail with "Unsupported portable setup version {v}. Hint: Upgrade apm."
       - Use: `use flate2::read::DeflateDecoder;`

       **`from_state_and_config(state: &InstallState, config: &Config) -> PortableSetup`:**
       - Build `PortablePlugin` for each `state.plugins` entry: `n=name, v=version, p=pinned, s=source`
       - Build sources from `config.sources` (the SourceEntry vec, NOT from config.sources() which includes official): each `(entry.name, entry.url)`
       - Build `PortableConfig`:
         - `sc`: `Some("system".to_string())` only if `config.install_scope == InstallScope::System`, else `None`
         - `reg`: `Some(config.default_registry_url.clone())` only if it differs from `DEFAULT_REGISTRY_URL`, else `None`
       - Set `v: 1`
